### 记录 04

**【开发阶段】** 里程碑 M2：音频核心初始化模块（对照计划书 Week8～Week9 前半）

**【发给 AI 的提示词原文】**

```
接下来按项目计划书，实现音频核心初始化模块：
1.  实现麦克风权限授权逻辑，授权成功获取音频流，拒绝/授权失败时给出明确文字提示；
2.  初始化AudioContext、AnalyserNode音频分析节点，配置FFT大小为1024；
3.  封装获取实时时域数据、频域数据、计算平均音量、主导频率的通用方法；
4.  做好浏览器兼容性判断，对非Chrome/Edge浏览器给出提示；
5.  未授权时禁用开始涂鸦按钮，授权成功后按钮可点击。
请直接修改项目文件，给出可运行的完整代码。
```

**【实现成果】**

1. **`BrowserSupport` 模块**
   - `check()`：识别 Chrome / Edge。
   - `hasMediaDevices()`：检测 `getUserMedia` 能力。
   - 非目标浏览器：页头警告 + 状态栏错误 + 全部按钮禁用。

2. **`AudioEngine` 模块**
   - `getUserMedia({ audio: true })` 麦克风授权。
   - 授权失败分类中文提示（拒绝、无设备、占用、安全限制等）。
   - `AudioContext` + `AnalyserNode`，`fftSize = 1024`。
   - 封装方法：
     - `getTimeDomainData()` — 时域数据（1024 点）
     - `getFrequencyData()` — 频域数据（512 bins）
     - `getAverageVolume()` — RMS 音量（0～1）
     - `getDominantFrequency()` — 主频 Hz
   - `dispose()`：释放麦克风流与音频上下文。

3. **启停流程**
   - 「开始涂鸦」：授权成功后自动启动可视化循环。
   - 「停止涂鸦」：停止 `requestAnimationFrame`、关闭麦克风。
   - 调试接口：`window.SoundDoodle`。

4. **按钮状态**
   - 未授权：仅「开始涂鸦」可点；停止/清空/保存禁用。
   - 运行中：「开始」禁用；「停止/清空/保存」可点。

**【当前项目进度说明】**

- 完成度：M2 验收项通过；频域 API 已就绪供 M4 使用。
- 里程碑：M0 ✅、M1 ✅、M2 ✅。
- 未完成：波形/频谱可视化绘制、交互涂鸦、保存 PNG。
- 下一步：M3/M4 声波可视化（波形与频谱）。

### 记录 05

**【开发阶段】** 里程碑 M3：实时声波波形曲线（对照计划书 Week9，计划书视觉元素第 1 项）

**【发给 AI 的提示词原文】**

```
现在实现第一个核心可视化效果：实时声波波形曲线。
1.  在Canvas上绘制以水平中线为基准的示波器风格波形，使用荧光绿线条；
2.  读取AnalyserNode的时域数据，映射为Y轴上下偏移的波形；
3.  用requestAnimationFrame逐帧渲染，保证延迟≤100ms；
4.  深色画布背景，波形流畅无卡顿、无断层；
5.  静音时波形静止，发声时跟随音量实时变化。
请修改代码并验证功能。
```

**【实现成果】**

1. **新增 `WaveformRenderer` 模块**
   - 颜色 `#39ff14`，线宽 2px，圆角端点，荧光绿发光效果。
   - Y 轴映射：`y = 中线 + (value - 128) / 128 × 振幅`（128 为静止中线）。
   - 静音判定：RMS &lt; 1.2% 时绘制水平静止直线。
   - 发声时：1024 点时域数据连续 `lineTo` 绘制。

2. **渲染循环优化**
   - `requestAnimationFrame` 主循环，每帧：读时域 → 清背景 → 绘波形。
   - `analyser.smoothingTimeConstant` 调整为 0.65，提升响应速度。
   - 状态栏 250ms 节流更新，显示「静音/发声」与音量百分比。
   - 窗口 `resize` 时若运行中立即重绘一帧。

3. **移除 M2 调试 HUD**，以波形为画布主视觉。

**【当前项目进度说明】**

- 完成度：计划书四类视觉元素第 1 项 ✅；整体功能约 35%。
- 里程碑：M0 ✅、M1 ✅、M2 ✅、M3 ✅。
- 未完成：频谱柱（M4）、轨迹线（M5）、粒子（M6）、保存 PNG、全面测试与交付。
- 下一步：M4 频谱柱状图（64 根）。
